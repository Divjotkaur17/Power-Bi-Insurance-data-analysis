## 📌 Project Overview
This project is a comprehensive data analysis of an insurance dataset using **Power BI**. The goal of this dashboard is to provide actionable insights into customer behavior, claim frequencies, premium distributions, and overall profitability. It is designed to help stakeholders quickly identify risk factors and optimize policy offerings.

## 🎯 Key Objectives
* Analyze customer demographics to identify high-risk and low-risk groups.
* Track the frequency and total cost of insurance claims.
* Evaluate the profitability of different insurance policy types.
* Provide an interactive, easy-to-read visualization for quick decision-making.

## 🛠️ Tools & Technologies Used
* **Power BI:** For building the interactive dashboard and data visualization.
* **Power Query:** For cleaning, transforming, and preparing the raw data.
* **DAX (Data Analysis Expressions):** For creating calculated columns and custom measures.
* **SQL Server:** Used for extracting, querying, and performing initial data manipulation on the raw insurance dataset before importing it into Power BI.

## 📈 Key Insights Discovered
* **Demographic Trends:** The age group of ADULTS accounted for the highest number of claims.
* **Policy Profitability:** The TRAVEL policy type generated the most revenue with the lowest claim rate.
